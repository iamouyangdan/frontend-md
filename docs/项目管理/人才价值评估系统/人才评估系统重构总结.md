# 微前端

[__https://qiankun.umijs.org/zh/guide__](https://qiankun.umijs.org/zh/guide)

改造步骤：

1. 修改package.json的name字段。

1. 修改vue.config.js，增加configureWebpack: 

```text
const { name } = require('../package.json')
module.exports = {
  ...,
  configureWebpack: {
    output: {
      // 把子应用打包成 umd 库格式
      library: `${name}-[name]`,
      libraryTarget: 'umd',
      jsonpFunction: `webpackJsonp_${name}`
    }
  },
}

```

1. 修改main.js
